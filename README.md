## Conflicto y resolución
- Descripción: Por un malentendido se estubieron haciendo pull request solo a la rama develop desde la rama develop, igualmente se hizo un pull request de la rama rama-stefan hacia rama-stefan. Lo que provocó que al final del proyecto la rama main solo tuviera la información que se hizo en el primer pull request (Codigo inicial funcional del proyecto) lo que ocasionó que el codigo no funcione.
- Solución: Luego de revisar el codigo de todos los PR y lo que contenía el main se llego al origen del error, se solucionó haciendo dos PR uno de develop -> main y otro rama-stefan -> main, esto completó el codigo y lo dejó funcionando.
